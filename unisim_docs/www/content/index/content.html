Overview

The "UNISIM Virtual Platforms" is a research project willing to provide academia and industry with a simulation platform to ease hardware/software codesign, integration and validation.

The project has started within the UNISIM simulation environment. While UNISIM and its simulators are based on a fast implementation of SystemC, named FastSysC (signal coding style only, no threads), the UNISIM Virtual Platforms are based on industry standards IEEE1666TM, OSCI SystemCTM and OSCI SystemCTM TLM 2.0. The rationals beyond these technical choices is to achieve full system simulation and fast enough simulation to enable debugging software on a virtual platform while maintaining a good representativity of the simulated target.
