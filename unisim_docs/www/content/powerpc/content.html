PowerPC Virtual Platforms

Source codes

  • A user level PowerPC simulator with support of ELF32 binaries and Linux system call translation: ppcemu-1.0-1.tar.gz
  • A Full system "PowerMac G4 PCI" like machine simulator (MPC7447A/MPC107) with Linux boot support: ppcemu_system-1.0-1.tar.gz
  • An MPC7447A/MPC107 board simulator with support of ELF32 binaries and targeted for industrial applications: embedded-ppc-g4-board-1.0-1.tar.gz
  • A demo GPS application based on Linux kernel 2.6.27.35 and Busybox 1.15.3 for the full system "PowerMac G4 PCI" like machine simulator: gps.tar.gz

Precompiled

  • For Windows: a Full system "PowerMac G4 PCI" like machine (MPC7447A/MPC107) with Linux boot support: setup-ppcemu_system-1.0-1.exe

Running demos
