PowerPC Virtual Platforms

Source codes

  • UNISIM ppcemu, a user level PowerPC simulator with support of ELF32 binaries and Linux system call translation: download
  • UNISIM ppcemu-system, a Full system "PowerMac G4 PCI" like machine simulator (MPC7447A/MPC107) with Linux boot support: download
  • UNISIM embedded-ppc-g4-board, a MPC7447A/MPC107 board simulator with support of ELF32 binaries and targeted for industrial applications: download

Older versions of these simulators are still available for download here.
